# Angular 2 Quickstart

Based on https://angular.io/docs/ts/latest/guide/webpack.html

## Get started

```
git clone  --depth 1 https://github.com/tinydesk/quickstart-angular2.git
```

Run the development server:
```
npm start
```

Run the unit tests:
```
npm test
```

Run the production build:
```
npm run build
```
